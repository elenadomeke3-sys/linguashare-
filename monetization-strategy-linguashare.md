## 3. Model Freemium

### 3.1 Funnel

```
┌──────────────┐
│    VISIT     │  100%
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   SIGNUP     │  10% (10 użytkowników)
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   ACTIVATE   │  50% (5 użytkowników - first download)
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   RETAIN     │  40% (2 użytkowników - wracają po 30 dni)
└──────┬───────┘
       │
       ▼
┌──────────────┐
│   CONVERT    │  5% (0.5 użytkowników - płacący)
└──────────────┘
```

### 3.2 Conversion Targets

| Metric | Target | Benchmark |
|--------|--------|-----------|
| Visit → Signup | 10% | 5-15% |
| Signup → First download | 50% | 40-60% |
| Download → Retention (30d) | 40% | 30-50% |
| Free → Paid | 5% | 3-8% |

## 6. Unit Economics

### 6.1 Customer Acquisition Cost (CAC)

| Kanał | CAC (PLN) | Źródło |
|-------|-----------|--------|
| Organic (FB Groups) | 0 | Community |
| Content SEO | 5 | Blog |
| Paid FB/IG | 25 | Ads |
| Referral | 10 | Program |

**Średni CAC:** 15-20 PLN

### 6.2 Lifetime Value (LTV)

| Metric | Wartość |
|--------|---------|
| ARPU (monthly) | 33 PLN |
| Churn (monthly) | 5% |
| LTV | 33 ÷ 0.05 = 660 PLN |
| LTV/CAC | 660 ÷ 20 = 33x |

### 6.3 LTV:CAC Ratio

| Ratio | Interpretation |
|-------|---------------|
| > 3:1 | ✅ Healthy business |
| 1:1 - 3:1 | 🟡 Need improvement |
| < 1:1 | ❌ Not sustainable |

**LinguaShare Target:** > 3:1 ✅

## 11. Wnioski

**Kluczowe zasady monetyzacji:**

1. **Freemium first:** Acquisition before monetization
2. **Simple pricing:** Tylko 3 plany (Free, Pro, Team)
3. **PL pricing:** 39 PLN = $9.99 = odpowiednie dla PL
4. **Focus on retention:** Higher LTV > higher conversion
5. **Scale slowly:** B2B i advertising = v2+

**Projekcja Year 1:** ~60k PLN ARR (conservative)
